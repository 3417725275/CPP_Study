fgets 是C语言中用于从文件流（通常是文本文件）中读取一行文本的标准库函数。它的原型如下：

char *fgets(char *str, int size, FILE *stream);

str：这是一个指向字符数组（或字符串）的指针，用于存储从文件中读取的文本。你需要提供足够大的数组来容纳读取的文本，以防止溢出。通常，你可以使用 char 数组来存储文本，例如 char buffer[1024]。

size：这是要读取的最大字符数，包括终止 null 字符（'\0'）。这个参数用于防止读取过多的字符，以防止缓冲区溢出。

stream：这是一个指向已打开文件的指针，从中读取文本。通常，你可以使用 stdin 表示标准输入，或者使用 FILE 类型的指针来表示你自己打开的文件。

fgets 函数会从文件流中读取一行文本，包括换行符 \n，并将其存储在 str 指向的字符数组中，直到达到指定的字符数（size - 1）或者遇到换行符为止。然后，它会在存储的文本后添加一个 null 字符 '\0'，以确保字符串以 null 结尾。

fgets 返回 str 指向的字符数组的指针，如果成功读取一行文本，则返回 str，否则返回 NULL，通常用于检查是否已经读取到文件的末尾或发生了错误。



fopen 是C语言标准库中的一个函数，用于打开文件。它的原型如下：
FILE *fopen(const char *filename, const char *mode);

filename：一个字符串，表示要打开的文件的名称或路径。可以是相对路径或绝对路径。如果文件不存在，fopen 通常会创建一个新文件，但这取决于打开模式。

mode：一个字符串，表示文件的打开模式，它指定了你打开文件的方式（读、写、追加等）。以下是常见的一些打开模式：

"r"：以只读方式打开文件。文件必须存在，否则打开失败。
"w"：以写方式打开文件，如果文件存在则截断为零长度，如果不存在则创建新文件。
"a"：以追加方式打开文件，如果文件存在则将数据追加到文件末尾，如果不存在则创建新文件。
"rb"：以二进制只读方式打开文件。
"wb"：以二进制写方式打开文件。
"ab"：以二进制追加方式打开文件。
fopen 函数返回一个指向 FILE 结构的指针，该结构表示打开的文件。FILE 结构包含了有关文件的信息，如文件指针、文件状态等。

```C++
#include <stdio.h>

int main() {
    FILE *file;
    char character;

    // 打开文件以只读方式
    file = fopen("example.txt", "r");

    if (file == NULL) {
        perror("Failed to open file");
        return 1;
    }

    // 读取文件中的字符并打印
    while ((character = fgetc(file)) != EOF) {
        putchar(character);
    }

    // 关闭文件
    fclose(file);

    return 0;
}
